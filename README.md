# Jarvis

Static GitHub Pages AI voice assistant inspired by Jarvis.

## Run

1. Enable GitHub Pages for this repository and set source to `/docs`.
2. Open the published page.
3. Enter your GitHub PAT (with access to GitHub Models) and select a model.
4. Click **Enable Devices** to grant microphone access and load speaker devices.
5. Hold **Push to Talk** to speak, then release to send.
