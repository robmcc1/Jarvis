# Jarvis

Static GitHub Pages AI voice assistant inspired by Jarvis.

## Run

1. Enable GitHub Pages for this repository and set source to `/docs`.
2. Open the published page.
3. Enter your GitHub PAT (with access to GitHub Models); the model list refreshes for that token.
   Then select a model.
   - Optional PAT memory is session-only (current tab), not persisted across browser restarts.
4. Click **Test Microphone** to grant microphone access and load speaker devices.
5. Hold **Push to Talk** to speak, then release to send.
